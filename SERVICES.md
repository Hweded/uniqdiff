# Services

`uniqdiff` Core is free and open-source under the Apache License 2.0.

Commercial services are available for teams that need help applying `uniqdiff` and
the broader UniqTools ecosystem in production.

## Available Services

- implementation support;
- production integration;
- performance audits;
- CI/CD workflow setup;
- custom connectors;
- row-level diff workflows;
- reporting workflows;
- private support;
- enterprise onboarding.

## Scope

Basic comparison, local files, CLI usage, exact backends, file result mode, lazy
readers, and the public engine API remain free in `uniqdiff` Core.

Paid services and commercial tools should live outside the `uniqdiff` engine package.
