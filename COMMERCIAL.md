# Commercial Use and Support

`uniqdiff` Core is free and open-source under the Apache License 2.0.

The core comparison engine, local file support, CLI, documented public API, result
objects, lazy result readers, and exact backends remain open-source. Basic comparison
features are not paid features.

## Commercial Offerings

Commercial offerings are provided around the engine through the broader UniqTools
ecosystem:

- implementation support;
- production integration;
- performance audits;
- CI/CD workflow setup;
- custom local, cloud, database, and enterprise connectors;
- row-level diff tooling;
- HTML/PDF/Excel reporting;
- private support and enterprise SLA.

Paid features should live outside the `uniqdiff` core package as separate UniqTools
products, commercial extensions, or service engagements.

## What Stays Free

- `compare`, `diff`, `unique`, `intersection`, and `duplicates`;
- local CSV, TSV, JSONL, TXT, gzip, and optional Parquet support;
- memory, SQLite, hash partitioning, external sort, and auto modes;
- file result mode and lazy result readers;
- connector protocol;
- direct `uniqdiff` CLI for scripts and CI/CD.

## What May Be Commercial

- `uniqreport` reports;
- `uniqrowdiff` field-level row changes;
- `uniqtools-cli` workflow orchestration;
- advanced CI/CD policy packs;
- custom connectors;
- private support;
- enterprise onboarding and integration.
