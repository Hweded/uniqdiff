# Support

## Community Support

Community support is available through public repository issues and discussions.

Use public issues for:

- reproducible bugs;
- documentation problems;
- small feature requests;
- compatibility questions;
- CLI or API usage questions.

Please include:

- `uniqdiff` version;
- Python version;
- operating system;
- a minimal reproducible example;
- expected and actual behavior.

## Security Issues

Do not report security issues in public issues. See [SECURITY.md](SECURITY.md).

## Commercial Support

Commercial support is available for teams that need:

- production integration;
- performance audits;
- private troubleshooting;
- CI/CD workflow setup;
- custom connectors;
- reporting or row-level diff workflows;
- enterprise support terms.

Commercial support is provided outside the open-source core package and does not
change the Apache-2.0 license of `uniqdiff` Core.
